This repository also includes a program to test your code. Be sure your class names and file names match otherwise the tester will not be able to find your programs.
If your output does not match the expected output text files exactly, the tests will fail. 
However, determining why these tests fail can be very tricky because of the whitespace involved. 
You can use this [website](https://text-compare.com/) to help compare the actual to expected output by using copy and paste from the results under the Actions tab.
Leave a message in this README if you are unable to get the program to pass the tester but the output looks virtually identical. 
Do **NOT** spend ages trying to debug the output for whitespace.

Recall that you can use the Preview feature when editing the README to see how it will look before you commit.
